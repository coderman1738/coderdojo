You should set your header bar to position: fixed instead.

Example:

div.header {
  position: fixed;
  top: 0;
  left: 0;
  right: 0;
}

if you are concerned about the top of .home being cut off, you can also add a margin-top

http://jsfiddle.net/R9Lds/1/
